# Простой Telegram-бот с меню и ссылками

## Что делает
- После `/start` показывает меню кнопками.
- Пользователь выбирает пункт.
- Бот отправляет ссылку, привязанную к выбранному пункту.
- Ведет суточную статистику уникальных пользователей в SQLite.
- Команда `/stats` доступна только `@Nazerati_06`.

## Настройка
1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Установите переменную окружения `BOT_TOKEN`:
   ```bash
   export BOT_TOKEN="ваш_токен"
   ```
4. Запустите:
   ```bash
   python bot.py
   ```

## Изменение пунктов меню
Отредактируйте словарь `MENU_LINKS` в `bot.py`:

```python
MENU_LINKS = {
    "Зеркало": "https://lkhq.cc/3ddb3d",
}
```
